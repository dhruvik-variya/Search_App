## ✨ Features

- **⚡ Lightning Fast Search:** Powered by Algolia, delivering instant results as you type.
- **📱 Fully Responsive:** Beautiful UI that looks perfect on both desktop and mobile devices.
- **🎨 Modern Design:** Styled with Tailwind CSS for a sleek, clean, and modern look.
- **🔍 Advanced Filtering:** Refine your search using hierarchical categories, brand filters, and price ranges.
- **📊 Real-time Stats:** Instantly see the number of results and search duration.
- **💯 Pagination:** Seamlessly navigate through multiple pages of search results.

## 🛠️ Tech Stack

- **[React 18](https://reactjs.org/)** - The library for web and native user interfaces.

## 🚀 Getting Started

Want to run this project on your local machine? Follow these simple steps:

### 1️⃣ Clone the repository
```bash
git clone https://github.com/dhruvik-variya/Search_App.git
cd "Search App Algolia"
```

### 2️⃣ Install dependencies
```bash
npm install
```

### 3️⃣ Start the development server
```bash
npm run dev
```

Your app should now be running on `http://localhost:5173/` 🎉

## 💡 How it works

The app uses `react-instantsearch` components to connect directly to an Algolia index.
- `<SearchBox>` handles the search input.
- `<Hits>` displays the individual product cards.
- `<RefinementList>` and `<HierarchicalMenu>` allow users to filter the products.
- `<Pagination>` manages page changes.

All styles are implemented through **Tailwind CSS** directly in the components, ensuring a consistent and responsive design across the app.
